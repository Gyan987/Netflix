# 🎬 Netflix Clone (Static Website)

A **static clone of the Netflix landing page** built using **HTML and CSS**.  
This project recreates the **Netflix homepage UI** to practice frontend layout, styling, and responsive design.

The website focuses purely on **design and layout**, without any backend functionality or JavaScript logic.

---

# 📌 Project Overview

This project is a **frontend UI clone** of Netflix's landing page.  
It replicates the look and feel of the official Netflix website using only **HTML and CSS**.

The goal of this project is to improve skills in:

- HTML page structure
- CSS layouts
- Responsive design
- Flexbox & positioning
- Modern UI styling

---

# ✨ Features

### 🎥 Netflix Landing Page UI
- Netflix style homepage layout
- Background hero banner
- Movie/series poster collage
- Netflix branding

### 📧 Signup Section
- Email input field
- "Get Started" button
- Styled similar to Netflix homepage

### 🧭 Navigation Bar
- Netflix logo
- Language selector
- Sign In button

### 📱 Responsive Design
- Layout adapts to different screen sizes
- Mobile-friendly structure

### 🎨 Visual Design
- Dark theme
- Netflix color scheme (Red + Black)
- Clean typography
- Modern layout
